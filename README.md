## Description

[Nest](https://github.com/nestjs/nest) framework dépôt de démarrage TypeScript.

## Installation

```bash
$ npm install
```

## Exécution de l'application

```bash
# développement
$ npm run start

# mode surveillance
$ npm run start:dev

# mode production
$ npm run start:prod
```

## Tests

```bash
# tests unitaires
$ npm run test

# tests e2e
$ npm run test:e2e

# couverture des tests
$ npm run test:cov
```
